# Diamond Price Prediction

## Problem
Predict diamond prices using machine learning.

## Workflow
- Data cleaning
- EDA
- Feature engineering
- Model training
- Model export (.pkl)
- Streamlit deployment

## Project Structure
data/          -> dataset  
notebooks/     -> EDA + training  
model/         -> trained model  
app/           -> Streamlit app  

## Run Locally
pip install -r requirements.txt  
streamlit run app/app.py  

## Output
Predicts diamond price from user input.
